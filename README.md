# snake-cpp

How to setup:
```
$ git clone https://github.com/KularatnaS/snake-cpp.git
$ cd snake-cpp
$ git submodule update --init --recursive
$ cmake -S. -Bbuild
$ cmake --build build
```

How to run:
```
$ ./build/snake
```
